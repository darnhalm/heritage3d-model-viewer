Где находится UI

  • Центральная полоска кнопок: #popup-buttons-parent внутри #popup → src/ui/popup-panel/index.tsx (PopupButtonControls, рядом AnimationControls). Стили: src/style.scss (left: 50%, bottom: 20px, горизонтальный flex).
  • Всплывающие панели к ним: #popup → PopupPanelControls в том же файле + src/ui/popup-panel/panels.tsx. Переключение через ui.active: 'info' | 'measurement' | 'view' | 'id' (и null, если панель закрыта).
  • Поля Camera / Sky / Light / Settings (сцена): вкладка Settings левой панели — src/ui/left-panel/index.tsx (CameraPanel, SkyboxPanel, LightPanel, SettingsPanel и др.).
  • Пустая сцена: у #popup класс empty — центральный блок скрыт.

Центральный навигационный блок (#popup-buttons-parent)

Ряд элементов по центру внизу канваса (#canvas-wrapper): left: 50% и transform, bottom ≈ 20px в style.scss. Контейнер #popup-buttons-parent внутри #popup (popup-panel/index.tsx). При клике по части кнопок открывается всплывающая панель над полоской (ui.active); клик вне #popup закрывает панель.

Содержимое панелей и выпадающих блоков

Панели: src/ui/popup-panel/panels.tsx; анимация — animation-controls.tsx. Блок над центральной полоской, не отдельное окно ОС. PCUI Select открывает список внутри этой панели.

Animation (в одной строке с кнопками, не ui.active):

  • Play / Pause — .anim-control-button, animation.playing.
  • Трек — #anim-track-select, animation.selectedTrack (список из animation.list).
  • Скраб — #anim-scrub-slider, animation.progress (останавливает воспроизведение).
  • Скорость — #anim-speed-select, 0.25x–2x → animation.speed.

Info (ui.active === 'info'):

  • Вкладки .info-tab: Controls | Model | About (часть скрывается в embed).
  • Controls → подвкладки .info-subtab: Desktop | Touch — подсказки Orbit/Fly; в Desktop при разрешении embed — General (клавиши F/R и inline-кнопки кадра/сброса камеры).
  • Model — статистика, Variant, дерево Hierarchy (scene.selectedNode.path), Show performance stats (debug.stats), Morph targets.
  • About — версия, кредиты, ссылки.

Measurement (ui.active === 'measurement'):

  • Toggle Measure Mode — measure.enabled.
  • Units (mm/cm/m) — measure.unit.
  • 1 Unit = (m) — measure.unitScale.
  • Known distance — measure.knownDistance; кнопка RECALCULATE SCENE SIZE.
  • Last Distance, Points; CLEAR MEASUREMENT.

ID (#id-panel, ui.active === 'id'):

  • Path, Name, Copy path.
  • Секция Metadata (Dublin Core) — заполненные поля из metadata.

View & share (#view-panel):

  • Embed Type Responsive / Fixed; UI Preset Full / Compact / Minimal.
  • Width (при Fixed), Height.
  • Тумблеры: левая панель, autoplay, fullscreen, POI, tour, measure, info, model info, controls, fit, reset.
  • Language auto / EN / RU / ZH.
  • Текстarea с кодом iframe, #copy-embed-button.
  • TAKE A SNAPSHOT AS PNG, COVER IMAGE (1:1), EXPORT VIEWER SETTINGS.

5. Embed Mode

Активируется параметром URL ?embed=true.

Пресет Описание
full Полный интерфейс
compact Скрыт ряд кнопок
minimal Только canvas + play

Параметры: panel, poi, tour, measure, info, modelInfo, controls, fit, reset, fullscreen, autoplay, ui=compact|minimal|full
